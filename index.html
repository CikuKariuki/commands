Installing Rabbit MQ

  • sudo apt-get update
  • sudo apt-get upgrade

    • Install Erlang: Erlang is a functional, general-purpose, concurrent programming language which was built for concurrency, fault tolerance and distributed application architectures

  • wget -O- https://packages.erlang-solutions.com/ubuntu/erlang_solutions.asc | sudo apt-key add -
  • echo "deb https://packages.erlang-solutions.com/ubuntu bionic contrib" | sudo tee /etc/apt/sources.list.d/rabbitmq.list
  • sudo apt update
  • sudo apt -y install erlang
  • Verify your installation of Erlang: erl
  • crtl + C twice to quit the erlang shell
  • Add the Apt repository to your Apt source list directory echo "deb https://dl.bintray.com/rabbitmq/debian xenial main" | sudo tee /etc/apt/sources.list.d/bintray.rabbitmq.list
  • curl -fsSL https://github.com/rabbitmq/signing-keys/releases/download/2.0/rabbitmq-release-signing-key.asc | sudo apt-key add - This will instruct apt to trust packages signed by that key.
  • sudo apt-get update
  • sudo apt-get install apt-transport-https so as to be able to download rabbitmq from bintray
  • sudo apt-get install rabbitmq-server -y --fix-missing install the rabbitmq-server package

    • Start the server:

  • sudo systemctl start rabbitmq-server.service
  • sudo systemctl enable rabbitmq-server.service
  • checking the status of RabbitMQ: sudo rabbitmqctl status

    • By default, rabitmq creates a user named "guest" so to create your own administrator account:

  • sudo rabbitmqctl add_user ciku password
  • sudo rabbitmqctl set_user_tags ciku administrator
  • sudo rabbitmqctl set_permissions -p / ciku ".*" ".*" ".*"

    • Enable the rabbitmq management console so you can monitor the rabbitmq server processes from a web browser

  • sudo rabbitmq-plugins enable rabbitmq_management
  • sudo chown -R rabbitmq:rabbitmq /var/lib/rabbitmq/

    • To setup an administrator user account for accessing RabbitMq server management console repeat:

  • sudo rabbitmqctl add_user ciku password
  • sudo rabbitmqctl set_user_tags ciku administrator
  • sudo rabbitmqctl set_permissions -p / ciku ".*" ".*" ".*"
